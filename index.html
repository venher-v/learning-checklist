HTML & CSS

Чекліст для зображень на лендінгу:

  • ✅ Іконки — тільки SVG
    • ➔ Оптимізувати через SVGOMG
  • ✅ Фото, банери, фони — WebP (або AVIF)
    • ➔ Оптимізувати через Squoosh або TinyPNG
  • ✅ Прозорі зображення — WebP (або PNG, якщо треба ідеальна якість)
  • ✅ Зображення великого розміру — одразу кілька версій (мобайл, планшет, десктоп)
    • ➔ Використовувати тег <picture>
  • ✅ Назви файлів — тільки англійською і без пробілів
    • Наприклад: hero-banner.webp, а не банер головна.png
  • ✅ Альтернативний текст (alt) — обов'язково
  • ✅ Завжди перевіряти розмір картинок перед завантаженням
